# PayrollWinFormsCpp

## Описание проекта
Данное приложение представляет собой систему расчёта заработной платы,
разработанную на языке **C++/CLI** с использованием **Windows Forms**.

Программа предназначена для учёта различных типов работ, расчёта итоговой
оплаты с учётом надбавок, сортировки данных и вычисления средней заработной платы.

В качестве хранилища данных используется **реляционная база данных SQLite**.

---

## Функциональные возможности
- добавление, редактирование и удаление типов работ;
- расчёт итоговой заработной платы с учётом процентной надбавки;
- сортировка по названию и по итоговой оплате;
- вычисление средней заработной платы;
- разграничение прав доступа (администратор / пользователь);
- сохранение данных в базу данных SQLite;
- импорт и экспорт данных в текстовый/CSV файл.

---

## Используемые технологии
- **C++/CLI**
- **Windows Forms**
- **SQLite 3**
- **STL (vector, string, smart pointers)**
- **PlantUML** (для диаграммы классов)

---

## Архитектура проекта
Проект построен по многоуровневой архитектуре:

1. **Бизнес-логика**
   - `PayrollDepartment`
   - `WorkTypeBase`
   - стратегии расчёта бонусов (`IBonusStrategy`)
   - собственные классы исключений

2. **Слой работы с базой данных**
   - `NativeDb` — нативная работа с SQLite
   - `DbManagerCLI` — C++/CLI-обёртка для взаимодействия с GUI

3. **Пользовательский интерфейс**
   - `MainForm`
   - `EditForm`
   - `LoginForm`
